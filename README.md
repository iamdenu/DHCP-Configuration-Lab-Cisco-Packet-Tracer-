# DHCP-Configuration-Lab-Cisco-Packet-Tracer-
A Cisco Packet Tracer lab demonstrating the implementation of a DHCP server on a Cisco Router. This project covers IP address pooling, address exclusion, and automated client configuration to optimize network management and scalability.


This project demonstrates the implementation of Dynamic Host Configuration Protocol (DHCP) within a local area network using Cisco networking equipment. The goal was to automate the IP addressing process for end devices, ensuring efficient address management and preventing IP conflicts.

🛠️ Network Components
Server: 1x Server PT (Acts as the DHCP Server)

Switch: 1x Cisco 2960 

End Devices: 3x PCs, 1x Server

Cabling: Copper Straight-Through



🧪To verify the setup, I performed the following:

DHCP Request: Set PC IP configuration to "DHCP" and confirmed a successful lease.

Command Line: Ran ipconfig on end devices to verify the assigned IP, Subnet, and Gateway.
